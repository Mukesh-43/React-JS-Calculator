# React-JS-Calculator

## Hosted Link: https://calculator-reactjs-e7c04.web.app/

## Created a simple calculator app with the features of
- Add
- Multiply
- Subtract
- Divide
- Clear
- Equals
- Add/multiply,subtract/divide is clubbed together (just like a normal calculator on our phone).
